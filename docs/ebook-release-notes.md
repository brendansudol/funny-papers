# Ebook release notes

The repository includes two ready-to-read EPUB editions. They are generated
from the public, versioned library and contain no PDFs, full-text paper
Markdown, private sources, extracts, manifests, or pipeline logs.

## 2026-07 — Initial edition

- [Field Guide](../dist/humor-and-llms-field-guide.epub) — the curated guide as
  a continuous read.
- [Complete Edition](../dist/humor-and-llms-complete.epub) — the field guide
  followed by 115 paper summaries, grouped by guide section.

Coverage follows the July 2026 catalog: 116 entries total, with summaries for
the 111 converted entries and four consulted restricted sources. T3 remains
unavailable, and T5 currently covers Suls but not Oring. Restricted works are
represented only by derived summaries, access notices, and official publisher
links. The evidence-certification limits described in
[`papers/DISCREPANCIES.md`](../papers/DISCREPANCIES.md) still apply.

Both files were built with Pandoc 3.10 and passed EPUBCheck 5.3.0 with no errors
or warnings. Unchanged source inputs produce byte-for-byte identical files.
