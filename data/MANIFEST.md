# Dataset Library Manifest

Public datasets released by papers in [humor-and-llms-field-guide.md](../humor-and-llms-field-guide.md). Regenerated 2026-07-13 by `scripts/build_data_manifest.py` from [datasets.json](datasets.json) (machine-readable source of truth). Sibling library: [papers/MANIFEST.md](../papers/MANIFEST.md).

**49 datasets**: 38 downloaded, 1 download FAILED, 2 gated (login/agreement required), 3 over size cap, not vendored, 5 no public release. ~9.3 GB on disk.

Layout: each dataset lives in `data/<key>/`, exactly as published (shallow git clone, HuggingFace snapshot, or direct download — see `fetch method`). Re-run `scripts/download_datasets.py` after editing datasets.json; it skips anything already present.
